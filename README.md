# SAP Website

A modern and responsive website built for SAP.

## Technology Stack
- HTML5
- CSS3 / SCSS
- JavaScript
- PHP (for mailer)

## Structure
- `/assets`: Contains all CSS, SCSS, JavaScript, fonts, and images.
- `*.html`: Page templates for the website.
- `mailer.php`: Server-side script for handling contact forms.

## Usage
Simply open the HTML files in your browser or host them on any static web server (like Apache, Nginx, or GitHub Pages) that supports PHP for the mailer.
